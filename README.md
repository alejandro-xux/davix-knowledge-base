# Davix Knowledge Base

Base de conocimiento funcional del sistema ERPX Davix.

## Acerca de esta documentacion

Este repositorio contiene la documentacion funcional y de negocio del sistema Davix, incluyendo:

- **Procesos de negocio**: Flujos operativos y reglas del sistema
- **Casos de uso**: Descripcion funcional de cada modulo
- **Guias de usuario**: Manuales paso a paso
- **Glosario**: Terminologia y conceptos clave

## Estructura

La documentacion esta organizada por modulos funcionales del sistema. Navega usando el menu lateral o consulta el [SUMMARY](SUMMARY.md) para ver el indice completo.
